# counting-valleys

A valley is defined as going lower than sea level and then back to sea level

### Input 
```
N = 8
S = UDDDUDUU
```
### Output
```
1

_/\      _
   \    /
    \/\/
```

### Input 
```
N = 10
S = UDDDUDUUDU
```
### Output
```
2

_/\      _  _
   \    / \/
    \/\/

```
